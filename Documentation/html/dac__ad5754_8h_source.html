1 /********************************************************************************
2  *
3  * dac_ad5754.h - headers for the ADI ad5754 DAC.
4  * Interfaces the TI tiva C Launchpad with the ADI dac5754 digital to analog
5  * converter
6  *
7  * Uses SSI0 on the TI Tiva C Series TM4C123G LaunchPad
8  * Uses SPI protocol for the serial interface to the DAC
9  *
10  * Author: Curtis Mayberry
11  * Georgia Tech IMEMS
12  * February 2014
13  *
14  * Originally written for the MRIG uHRG gyroscope project
15  *
16  * INPUTS
17  * ~LDAC_Forcer
18  * ~LDAC_Quad
19  * ~CLR
20  * SERIAL (SPI)
21  * SCLK
22  * ~SYNC
23  * SDO
24  * SDIN
25  * OUTPUTS
26  * All 4 outputs of each DAC may be used
27  *
28  * Transfers over SSI to the DAC can be handled by the CPU or by the uDMA.
29  *
30  * uDMA Mode: In order to use the uDMA mode, DAC_UDMA_MODE must be defined in the
31  * main program. In TI CCS this can be done by setting a predefined symbol in
32  * the project properties menu. The current implementation only handles software
33  * initiated transfers in uDMA auto mode. Two buffers can still be used in a
34  * ping-pong configuration (not ping-pong mode).
35  *
36  * This work is licensed under the Creative Commons Attribution-ShareAlike 3.0
37  * Unported License. To view a copy of this license, visit
38  * http://creativecommons.org/licenses/by-sa/3.0/ or send a letter to Creative
39  * Commons, 444 Castro Street, Suite 900, Mountain View, California, 94041, USA.
40  *
41  ********************************************************************************/
